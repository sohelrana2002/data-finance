# 💰 Data Finance Dashboard

A modern and responsive personal finance dashboard built with **React**, **Vite**, and **Pure CSS**. This project helps users manage income, expenses, savings, and financial analytics through a clean and interactive user interface.

---

## 🚀 Live Demo

- **Website**: [View Live Website](https://sohelrana2002.github.io/data-finance/)

---

## ✨ Features

- 📊 Interactive Finance Dashboard
- 📈 Financial Statistics & Charts
- ⚡ Built with Vite for fast performance
- ⚛️ Component-based React architecture
- 🎨 Styled using Pure CSS
- 📱 Fully Responsive Design
- 📉 Expense Visualization

---

## 🛠️ Technologies Used

- React.js

- Pure CSS

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sohelrana2002/data-finance.git
```

Go to the project folder:

```bash
cd data-finance
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
